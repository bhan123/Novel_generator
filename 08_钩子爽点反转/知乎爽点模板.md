## 模板一：小人物对阶层的愤怒呐喊

**典型案例**：《簪中录》38,502赞 - 吕至元杀公主为女复仇

### 结构拆解

| 要素 | 内容 |
|------|------|
| **前置压抑** | 读者看到主角女儿被权贵侮辱、被迫寻死，而加害者逍遥法外；父亲被描写成嫌弃女儿的混蛋 |
| **信息差** | 所有人都以为吕至元是贪财无耻的小人，只有读者逐渐发现：一切都是伪装，他只是在保护女儿 |
| **爆发方式** | 当真相揭露——父亲亲手杀公主，只为证明女儿无辜。整个朝堂震动，皇帝崩溃 |
| **见证者** | 满堂寂静的朝臣、皇帝郭淑妃、黄梓瑕 |
| **奖励反馈** | 女儿滴翠的冤屈被洗清；父亲的爱得到正名；读者积压的愤怒得到宣泄 |

### 可迁移题材
- 古代权谋文、宫斗文
- 现代都市底层逆袭
- 职场打脸文
- 任何涉及"弱者是好人"的反转场景

### 使用风险
- 容易写成纯虐，读者弃文
- 真相揭露太晚会流失读者
- 父亲动机要提前埋线，不能太突兀

### 可复用模板
> "蝼蚁般的小民发出对阶层不公的愤怒呐喊" → 最底层的父亲用最极端方式为女儿正名

## 模板二：底层成为历史转折点

**典型案例**：《将夜》7,434赞 - 农夫杨二喜成为大唐第一波反击

### 结构拆解

| 要素 | 内容 |
|------|------|
| **前置压抑** | 朝廷战报传来，举国震动，所有人都觉得大唐必败；一个农夫只认识几个字，被妻子嘲笑"大唐是你家的？" |
| **信息差** | 所有人都不知道这个农夫曾是边塞老兵，只有他自己知道 |
| **爆发方式** | 农夫默默离家，骑着小毛驴上了战场，用当年在边塞学的本事杀敌——成了大唐最开始的反击 |
| **见证者** | 妻子，村民，历史记载（但史书不会记得他名字） |
| **奖励反馈** | 小人物被历史记住；"大唐第一波反击来自农夫"这个认知反转 |

### 可迁移题材
- 穿越历史文
- 战争题材
- 末世求生文
- 任何"小人物改变大局"的场景

### 使用风险
- 容易写成自嗨，读者无感
- 铺垫要足够长，让读者认可这个人物
- 结局最好有历史记录的呼应

### 可复用模板
> "大唐最开始的反击不是来自将军，而是来自一个农夫"——小人物用沉默的行动改变历史

## 模板三：装逼打脸·认知反转

**典型案例**：《灵气逼人》4,074赞 - 曹大爷传授"淬体秘法"

### 结构拆解

| 要素 | 内容 |
|------|------|
| **前置压抑** | 读者和主角一起期待"神秘强大的修炼秘法" |
| **信息差** | 只有曹大爷知道所谓"最强淬体术"就是眼保健操和广播体操 |
| **爆发方式** | 曹大爷一本正经教"揉天应穴"，手机放出"眼保健操现在开始"——打脸来得猝不及防 |
| **见证者** | 楚歌等所有期待的年轻人 |
| **奖励反馈** | 认知被彻底颠覆；笑点+爽点同时爆发 |

### 可迁移题材
- 搞笑文、玩梗文
- 穿越都市文
- 任何有"高人装逼传授假秘法"的场景

### 使用风险
- 笑点文容易变毒点
- 反转要有逻辑，不能纯消遣
- 核心认知要正确：低配秘法确实有用，只是包装low

### 可复用模板
> "最强淬体秘法"其实是眼保健操——用最大期望值衬托最小实际收获的反差爽点

## 模板四：时间循环·蝴蝶效应

**典型案例**：《日食之影》3,751赞 - 循环中改变过去影响现在

### 结构拆解

| 要素 | 内容 |
|------|------|
| **前置压抑** | 主角陷入生死循环，每次在医院死去又穿越回十年前；读者和主角一起焦虑"怎么逃出循环" |
| **信息差** | 主角以为救了自己就能脱困，但心理老师点出：他潜意识删除了那一天的记忆 |
| **爆发方式** | 主角发现自己穿越回去做的所有事，都是在逃避那件"比活下去更重要的事" |
| **见证者** | 心理老师南梦浅 |
| **奖励反馈** | 读者对"为什么循环"有了更深理解；悬疑感+情感冲击 |

### 可迁移题材
- 悬疑推理文
- 穿越重生文
- 任何有"秘密被揭露"的情节
- 心理创伤类故事

### 使用风险
- 悬疑线要提前埋好，不能收不回来
- 核心秘密要有震撼感
- 情感线要真实

### 可复用模板
> "你穿越回去救自己，但你潜意识删除了那天真正重要的事"——循环不是逃生，而是面对被遗忘的真相

## 模板五：传承时刻·衣钵被授

**典型案例**：《从姑获鸟开始》3,377赞 - 老剑客杨三井传衣钵

### 结构拆解

| 要素 | 内容 |
|------|------|
| **前置压抑** | 主角自傲剑术大成，被老剑客一句"井底之蛙"打脸；读者也跟着尴尬 |
| **信息差** | 老剑客的真正实力只有读者通过主角视角感受到——那黑色的剑意浪潮让主角根本无力反抗 |
| **爆发方式** | 主角挨打后反思，重新讨教，用挨打换来的领悟反击——最终获得衣钵 |
| **见证者** | 滔滔江水边的两人 |
| **奖励反馈** | 主角获得剑道真传；"小成用眼，大成用刺"的领悟让主角突破 |

### 可迁移题材
- 武侠仙侠文
- 竞技文
- 任何"师徒传承"场景
- 装逼打脸升级流

### 使用风险
- 传承要有门槛，不能太容易
- 主角要有成长，不能纯挨打
- 师父要有高光时刻，不能沦为工具人

### 可复用模板
> "你剑术大成？井底之蛙"——用最大打击衬托最大机缘的反转型爽点

## 模板六：隐藏修为·反杀碾压

**典型案例**：《修真四万年》2,900赞 - 团长"忘记"问主角来历

### 结构拆解

| 要素 | 内容 |
|------|------|
| **前置压抑** | 主角因暴露修为被调查，所有人都在等团长追问；读者也跟着紧张 |
| **信息差** | 团长其实早就知道主角底细（蜘蛛巢星灵纹刺青），但选择不问 |
| **爆发方式** | 团长的回应："他来自哪里，关我屁事？"——实力碾压式的尊重 |
| **见证者** | 雇佣兵众人、军师白开心 |
| **奖励反馈** | 主角感受到被尊重；信息差让读者也跟着爽 |

### 可迁移题材
- 都市异能文
- 修仙文
- 任何有"隐藏身份"的场景
- 团队伙伴信任线

### 使用风险
- 团长人设要有厚度
- 不能太长，爽点要快速释放
- 信息差要合理，不能太刻意

### 可复用模板
> "我懒得问你的过去，正如你也从来没问过我"——用不在意衬托真正的高手风范

## 模板七：以弱胜强·信仰对抗

| 要素 | 内容 |
|------|------|
| **前置压抑** | 宗教骑士代表神圣正义，莱特只是普通人；骑士用圣光压迫弱者 |
| **信息差** | 只有读者知道：宗教早已腐败，圣光可以被罪恶使用 |
| **爆发方式** | 莱特徒手捏碎圣光屏障，质问"为什么你们这些罪人还能使用圣光" |
| **见证者** | 圣光骑士、贝尔克 |

### 使用风险

## 爽点模板核心规律总结

### 1. 信息差是爽点的灵魂
- 读者知道角色不知道 → 紧张感
- 角色知道读者不知道 → 悬念感
- 双方都不知道 → 好奇感
- 双方都知道了 → **爆发！**

### 2. 前置压抑决定爆发强度
- 压抑越久，爽点越强
- 压抑要有层次，不能一步到位
- 压抑期间要给读者"希望"

### 3. 见证者放大爽感
- 没人看到的装逼不是装逼
- 见证者的震惊反应是爽点的延伸
- 见证者要有层次（普通人→高手→敌人→爱人）

### 4. 爽点要可复制但不能重复
- 每个模板可以用多次
- 但每次要用不同的情节包装
- 内核相似，外壳要变
