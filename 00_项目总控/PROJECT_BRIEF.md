# 网络小说多 Agent 创作工厂

## 总目标
搭建一套网络小说工业化创作流程：题材研究 → 世界观 → 总纲 → 卷纲 → 细纲 → 钩子/爽点/反转 → 人物/对话 → 审核 → 可写作章节卡片。

## 核心原则
1. 不复制具体作品文本、人物、桥段，只抽象类型规律和情绪机制。
2. 所有输出都要沉淀为 Markdown 文件。
3. 大纲、细纲、爽点、人物、设定必须可追踪、可审核、可复用。
4. 审核 Agent 只审查，不替创作 Agent 找借口。
5. 所有素材分析结果必须转成"结构化卡片"。

## 目录约定
- 01_素材输入：外部材料、知乎链接、网文结构卡片
- 02_题材研究：题材规律、类型套路、开局模型
- 03_世界观：世界观圣经、等级体系、势力地图
- 04_人物：人物设定、人物关系、成长线
- 05_总纲：全书主线、长期矛盾、卷级目标
- 06_卷纲：每卷结构
- 07_细纲：章节级细纲
- 08_钩子爽点反转：爽点库、钩子库、反转库
- 09_对话风格：角色口吻、冲突对话模板
- 10_审核报告：设定、节奏、毒点、商业性审核
