## 一、世界观设定

### 1.1 双层宇宙架构

**太极界**：常规修真世界，遵循阴阳五行法则。自「虚月当空」后沦为太素界的缓冲带。

**太素界**：不可名状的混沌原初态，以「物非物」为基本粒子，物理法则呈现分形、递归、不可判定特征。

### 1.2 核心规则

- **认知即污染**：直视「道祖」会引发「临渊入梦」，本质是人类大脑被迫处理高维信息导致的神经重构
- **修行即献祭**：「炼炁」实为向太素界缴纳认知税，「金丹」是稳定的信息茧房，「元婴」是可移动的污染发射器
- **情绪即能量**：系统「心情值」对应精神稳定性阈值，数值越低越易被太素同化，但突破阈值后将获得「非人视角」

### 1.3 重要设定

| 设定 | 说明 |
|------|------|
| **虚月** | 两轮异色月亮，实为太素界在三维空间的投影切片，相位变化直接影响污染强度 |
| **不知面** | 墨竹山制式面巾，利用织物经纬线制造微弱认知盲区 |
| **墨山体操** | 暗合《太素四十九般变化》前七式的健身操，可同步调整脑波频率 |
| **五仙汤** | 喝五碗可升仙的汤药（消解传统仙侠崇高感） |

### 1.4 关键地点

- **墨竹山洞天**：八臂六首魔尊法坛支撑的折叠空间，时间流速1:3，寒潭水含高浓度玄冥真炁
- **朔方镇**：艮国边关要塞，城墙嵌三百六十枚「镇魂钉」，每枚刻不同版本《道德经》
- **禺山魔窟**：南宫家族地下基地，按《河图洛书》排布的巨型「认知迷宫」
- **昆仑墟**：已被太素化为发光菌毯

### 1.5 社会架构

三层嵌套结构：
1. **表层**：人间道王朝（离国、艮国等），科举制与修真资格绑定
2. **中层**：三大派（仙宫/玄门/神教）与散修联盟（墨竹山/娄观道）
3. **底层**：贤者公司主导的「量子网络」，悟道境以上修士被植入「偏差值监测模块」

## 二、主角与金手指

### 2.1 主角：李凡（道号李清月）

**穿越身份**：现代打工人，普通社畜

**核心矛盾**：「理性防御机制」与「神性侵蚀本能」的持续拉锯

**性格特征**：表面懒散油滑、实则心如明镜，以「打工心态」解构修行——把渡劫当KPI，将拜月作加班

### 2.2 心理健康维护系统（坑爹流金手指）

**心情值机制**：
```
心情值：0-100（唯一属性）
- 每次心理波动：抽奖充能+1
- 充能进度100/100：可抽奖一次
- 波动双向计算（上升+下降都有效）
```

**先天气运选项**（李凡被迫选择「镇静一点」）：
- 天塌不惊：大幅降低心情低落几率
- 乐天知命：心情最大上限提升，心情提升时数值翻倍
- **镇静一点**：心情波动时只变化1点

**系统功能**：
- 心理健康维护
- 抽奖功能（吃饼可恢复心情，芝麻饼+2点）
- 心理崩溃会持续掉心情（每秒-1点）

### 2.3 玄天剑意

北辰剑宗覆灭前斩出的最后一道剑意，非器灵而是「剑道人格化」。

- 性格暴烈偏执，视李凡为临时宿主兼复仇工具
- 传授《如意剑经》
- 「如指臂使」本质是让宿主神经突触与剑意共振，实为潜移默化的精神寄生

太素是「先天五太」之一，代表「质之始而成形」。

**道体修炼**：多形态进化与失控风险并存

**派系分歧**：
- **速成派**：主张「以魔制魔」，主动摄入低阶魔胎维持道体平衡（墨竹山）

## 四、篇章剧情梳理

### 第一卷：逍遥游（第一章 — 第七十八章）

#### 核心剧情

**第一章「观星」**：穿越开局

- 主角李凡穿越至墨竹山，被老道士收为弟子
- 墨竹山：幅员三千里，两山十四峰，妖兽横行，毒瘴密布
- 法坛配置：八臂六首魔尊法相，七纵七横四十九个蒲团
- 二十一名童子被选中，暗藏魔胎者送去「给山主下酒」

**双月之夜异变**：
1. 虚月当空 → 两轮非地球月亮的明月，一青一紫
2. 八臂魔尊托月，照映得六首一铁青一绛紫
3. 黑曜石金光烁烁，「好像睁开了数不尽的眼睛」
4. 黑风狂卷，环绕老道士画的圆圈
5. 时间消失，天地万物仿佛在此刻萧影无踪

**天降小剑**：
- 右手突然多出一枚三寸小剑
- 剑身无颚，剑脊闪烁星光
- 虚握即切开右掌，血线飙出
- 小剑如银鱼，钻进李凡掌心
- 系统触发抽奖，充能100/100

**血腥场景**：
- 老道士自挖双目，额头核桃大血洞
- 八臂六首魔尊被一剑劈成两半
- 童子尽皆消失
- 圈外散落：焦尸、干枯骨骸、七头八臂的脏器异化死胎

**核心悬念**：「物」与「非物」的判定标准，涉及生死

**场景描述**：李凡讲《道德经》时，听众因解读产生躯体畸变——裂开头颅、撕裂肋骨、化出肉翅、飞上天空

**原文金句**：
> "道可道，非常道；名可名，非常名。无名，天地之始，有名，万物之母。嗯，说不定还有个'名'，叫孕育万子千孙的森之黑山羊，呵呵……"
> "我悟道了！我悟道了！嚯嚯嚯哈哈哈哈嘎——嘎——嘎——！！！"
> "唉，道德经不能瞎讲了，又给整疯了一个……"

**爽点机制**：崇高（道德经）与恐怖（克苏鲁畸变）的反差冲击

## 七、创作特色总结

### 7.1 风格调性

- **学术论文体写荒诞设定**：用《为知法》解构认识论，以《巍巍不动泰山深根结果宝卷》模拟编程手册
- **黑色幽默**：称「魔胎」为「高蛋白食材」，把「渡劫」叫作「甲方验收」，将「悟道」戏称为「系统升级」
- **反讽张力**：李凡的吐槽常被证明是错误认知（不可靠叙述者结构）

### 7.2 叙事手法

- **章节命名系统**：「逍遥游」「平妖传」「风从虎」等七卷标题全部取自先秦典籍
- **卷内章节目录暗合《周易》卦象**：如「逍遥游」共78章，7+8=15，对应《谦卦》
- **元小说结构**：第483章切换至「张陵」视角，揭示整个故事是量子网络中的一个运行实例

### 7.3 设定考古学

所有「非科学」设定均有东方典籍锚点：
- 「虚星」出自《淮南子》「太素者，质之始也」
- 「真空家乡」化用白莲教经典
- 《太素四十九般变化》呼应《庄子·齐物论》
