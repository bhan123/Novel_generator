## 核心设定

### 世界观：养殖场景观

**核心概念：人类是被更高存在"养殖"的牲畜**

```
天坑事件 → 学校坠入未知森林
    ↓
走出森林 = 获得正式考验资格
    ↓
修炼至巅峰后发现：更高世界同样是被养殖的牧场
    ↓
"养殖真相比"——套娃设定，层层嵌套
```

**五大元素空间、九大规则贯穿全文**

**史前文明线**：
- 亚特兰蒂斯（第四纪元）
- 穆利亚文明（第三纪元）
- 美索不达米亚（第二纪元）

## 文风特征

**优点**：
- 前期心理描写细腻，恐惧感塑造出色
- 异能系统创意十足（隐形+符咒+变神+元素组合）
- 主角非狗屎运，凭勇气和战斗天赋
- **"高处不胜寒"**主题——力量越强内心越孤独

**缺点**：
- 后期异能变成单纯比拼力量，失去技巧性
- 格局过大，作者驾驭不足
- "养殖"真相用两次，惊讶感递减
- 史前文明虎头蛇尾
- 后宫线全部失败，主角孤独终老

**评分**：前500章8分神作级 / 中期400章4分 / 后200章1分

## 写作技巧

### 异能组合系统
```
基础异能 → 组合升级 → 战术多样性
隐形 + 符咒 = 暗杀
变神 + 元素 = 形态变化
```

### 套娃真相结构
```
第一层：人类被养殖
    ↓ 发现
第二层：养殖者也被养殖
    ↓ 发现
第三层：？？？（世界观无限扩大）
```

### 战力膨胀控制
问题：后期战力膨胀到星系级，战斗失去意义
教训：无限升级流需要提前设计战力天花板
