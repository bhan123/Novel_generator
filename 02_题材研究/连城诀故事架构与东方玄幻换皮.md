## 第一部分：原版《连城诀》故事架构

### 一、作品定位

| 项目 | 内容 |
|------|------|
| **原作** | 《连城诀》，金庸武侠小说 |
| **创作年代** | 1963年 |
| **题材** | 传统武侠 |
| **核心主题** | 人性贪欲、拜金主义、江湖险恶 |
| **基调** | 黑暗写实，揭示人性丑恶 |

---

### 二、核心设定

#### 2.1 宝藏设定

**连城诀/连城剑法**：
- 梁元帝留下的价值连城的珠宝
- 藏宝图隐于连城剑法口诀中
- 绝世武功 + 连城财宝，引得江湖中人趋之若鹜
- 口诀源自唐诗剑法

**宝藏所在地**：天宁寺

#### 2.2 武功体系

| 武功 | 来源 | 特点 |
|------|------|------|
| **连城剑法** | 梅念笙（原创）/ 梁元帝（口诀来源） | 剑招精妙，口诀藏宝图 |
| **神照功** | 丁典 | 绝世内功，可治愈重伤 |
| **血刀刀法** | 血刀门 | 邪派绝学，狠辣诡异 |
| **唐诗剑法** | 戚长发（歪曲传授） | 连城剑法的伪装版本 |

### 四、详细剧情线

#### 第一阶段：师门血案（前世纠葛）

**背景**：梁元帝一批价值连城的珠宝连同藏宝图下落不明，八百年后江湖上传闻藏宝图隐于连城剑法中。

**梅念笙**是唯一知道连城诀的人，一生只收过三个徒弟：万震山、言达平、戚长发。

| 事件 | 详情 |
|------|------|
| 三徒弑师 | 三弟子在得知师傅知晓连城剑法后，起贼心杀害师傅夺得剑谱，梅受重伤跃入江中 |
| 丁典救师 | 梅念笙被侠士丁典救起，在临终前将连城诀传给丁典 |
| 剑谱失踪 | 三徒争夺中剑谱不翼而飞，从此三徒互相跟踪寻找 |

#### 第二阶段：狄云成长与被害（第一章 - 第二十章左右）

**戚长发**在盗取剑谱后隐姓埋名乡下，收狄云为徒，女儿戚芳也与狄云青梅竹马。

| 事件 | 详情 |
|------|------|
| 歪曲教剑 | 戚长发怕女儿徒弟破解口诀与自己争夺秘籍，故意教错口诀 |
| 万府祝寿 | 万震山五十大寿，戚长发带狄云、戚芳赴宴 |
| 吕通寻仇 | 吕通与万震山有仇，在寿宴上闹事，狄云出手相助 |
| 老丐授招 | 一驼背老乞丐（实为言达平伪装）教狄云连城剑三招 |
| 万圭陷害 | 万圭垂涎戚芳，设计诬陷狄云入狱：偷盗剑谱、非礼女子 |
| 酷刑折磨 | 狄云受尽断指、穿琵琶骨酷刑 |
| 师妹误会 | 戚芳误会狄云，嫁给万圭 |

#### 第三阶段：丁典与凌霜华（平行线）

| 事件 | 详情 |
|------|------|
| 狱中相识 | 丁典身藏连城诀被凌霜华之父凌知府（凌退思）囚于狱中，与狄云成为生死之交 |
| 爱情悲剧 | 丁典和凌霜华热恋，凌霜华每日天台浇花暗示（菊花） |
| 神照功传 | 丁典将神照经传给狄云 |
| 越狱失败 | 丁典心中放不下凌霜华，与狄云一起越狱，不料霜华已死 |
| 丁典中毒 | 凌退思在女儿棺材中下毒，丁典中毒而亡 |

#### 第四阶段：雪山被困（经典段落）

| 事件 | 详情 |
|------|------|
| 血刀掳人 | 血刀老祖为得到连城诀围攻丁典，误认狄云为同门，掳走水笙 |
| 雪谷围困 | 血刀老祖携狄云和水笙逃入雪山谷，"落花流水"四侠追入 |
| 水岱惨死 | 水岱（花铁干、水笙之父）追杀血刀老祖，被杀 |
| 花铁干黑化 | 花铁干在粮尽时杀死水岱，后因恐惧向血刀老祖下跪求饶，暴露怯懦本性 |
| 血刀之死 | 狄云在激愤中无意间踢死血刀老祖 |
| 困居山洞 | 狄云和水笙被困在雪岩下山洞中半年 |
| 学会神功 | 狄云学会血刀心法 + 神照经内功，技盖江湖 |

#### 第五阶段：复仇与真相（结局）

| 事件 | 详情 |
|------|------|
| 万门恩怨 | 万震山、万圭为争夺宝藏彻底疯狂，父子互残 |
| 戚芳被杀 | 戚芳发现万圭歹毒行径，心软放万圭，被万圭杀害 |
| 狄云复仇 | 狄云杀死万圭和万震山 |
| 宝藏之毒 | 万震山、言达平、戚长发在互夺连城诀中相残而亡；众人因贪图宝藏中的毒物而死 |
| 结局 | 狄云带戚芳遗孤"空心菜"，与水笙归隐雪谷 |

### 五、核心情节线

```
【主线1：宝藏追逐】
连城诀口诀 → 众人争夺 → 宝藏现身 → 贪欲暴露 → 众恶相残

【主线2：狄云成长】
憨厚青年 → 被卷入阴谋 → 入狱受苦 → 学会神功 → 看透虚妄 → 归隐

【主线3：丁典与凌霜华】
狱中热恋 → 凌退思阻挠 → 越狱失败 → 双双惨死

【主线4：花铁干堕落】
侠客伪装 → 雪山被困 → 饥饿考验 → 杀友求荣 → 人设崩塌

【主线5：师门背叛】
梅念笙被弑 → 三徒互残 → 真相揭露 → 因果报应
```

### 四、换皮后核心剧情

#### 【换皮版】大纲框架

**世界背景**：仙魔大战后万年，天地灵气复苏，修仙界繁荣。一座上古仙府遗迹现世，据说藏有太古仙缘和"太上忘情道"核心传承，引得天下修士趋之若鹜。

#### 第一卷：凡尘卷（对应原版第一阶段）

**背景设定**：
- 陈凡，散修少年，自幼被苏家收养，与苏琳青梅竹马
- 苏家是仙朝小家族，三长老苏长老（戚长发）掌握一套"残缺传承"（原版唐诗剑法）
- 实际这套残缺传承来自上古仙府遗迹的口诀

**核心事件**：

| 原版 | 换皮后 |
|------|--------|
| 梅念笙被三徒弑师 | 上代宗主被三长老谋害，宗主临终前将"太上忘情道"传给散修萧云 |
| 丁典救师获得传承 | 萧云救下宗主，获得太上忘情道传承 |
| 三徒争夺剑谱 | 三长老争夺残缺传承，互相对峙 |

#### 第二卷：禁灵狱卷（对应原版第二阶段）

**核心事件**：

| 原版 | 换皮后 |
|------|--------|
| 万府祝寿 | 周天宗五十寿宴，苏家赴宴 |
| 吕通寻仇 | 周家仇敌上门闹事 |
| 老丐授招 | 赵先生（二长老）伪装老乞丐，传授陈凡三招"残缺传承" |
| 万圭陷害入狱 | 周子轩垂涎苏琳，设计陷害陈凡偷窃宗门秘典，陈凡被打入"禁灵狱" |
| 断指穿骨酷刑 | 禁灵狱中受尽折磨 |
| 师妹误会嫁仇人 | 苏琳误会陈凡，被迫嫁给周子轩 |

#### 第三卷：狱缘卷（对应原版第三阶段 - 丁典线）

**核心事件**：

| 原版 | 换皮后 |
|------|--------|
| 丁典狱中传功 | 萧云与陈凡在禁灵狱相识，将"神照经"传给陈凡 |
| 凌霜华热恋 | 萧云与林清婉（仙朝贵女）热恋 |
| 凌退思囚女 | 林清婉之父林尚书觊觎太上忘情道，将萧云囚禁，将女儿活埋 |
| 越狱失败/双双惨死 | 萧云与陈凡越狱失败，林清婉死亡，萧云中毒而亡 |

#### 第四卷：太古遗迹卷（对应原版第四阶段 - 雪山线）

**核心事件**：

| 原版 | 换皮后 |
|------|--------|
| 血刀老祖掳人 | 血刀魔君为夺太上忘情道围攻萧云，误认陈华为魔门弟子，掳走萧雨瑶 |
| 雪山被困 | 血刀魔君携陈凡、萧雨瑶逃入"太古遗迹" |
| 落花流水四侠追入 | 正道天骄"落花流水"追杀入遗迹 |
| 水岱惨死 | 水岱被血刀魔君杀害 |
| 花铁干堕落 | 韩师兄在遗迹中暴露本性，杀友求荣 |
| 血刀之死 | 陈凡在激愤中踢死血刀魔君 |
| 困居半年学会神功 | 陈凡困于遗迹深处，学会血刀魔功+神照经，实力大增 |

#### 第五卷：仙魔卷（对应原版第五阶段 - 复仇结局）

**核心事件**：

| 原版 | 换皮后 |
|------|--------|
| 万门恩怨 | 周天宗、周子轩为争夺太上忘情道彻底疯狂，父子互残 |
| 戚芳被杀 | 苏琳发现周家歹毒行径，心软放周子轩，被杀 |
| 狄云复仇 | 陈凡杀死周子轩和周天宗 |
| 宝藏之毒 | 众人因觊觎太古遗迹中的"禁忌灵物"中毒而死 |
| 因果报应 | 三长老在争夺中相残而亡 |
