## 核心创新点

### 题材类型
**暗黑权谋西幻**——脱胎于DND但不拘泥，自成体系

**与小白文的本质区别**：
- 不走升级打怪换地图的套路
- 核心冲突是**权谋博弈**而非战斗力碾压
- 主角是小人物，靠智慧和手段而非无敌实力

---

## 世界观设定

### 位面体系

```
天界：十二主神主导的最强位面
    ↓
人界（格罗里亚大陆）：力量上限21级，故事主位面
    ↓
异界（死亡世界）：亡灵世界，八位最高君王主导
    ↓
魔界：魔族位面
    ↓
深渊世界：强大深渊生物生存
```

**核心规则**：
> "真正的力量是制订规则"
> 世界分为三级：神（掌控位面、攫取信仰之源）→ 强者（受限于位面规则）→ 凡人（最底层，作为"食粮"）

## 修炼体系

| 等级 | 境界 |
|------|------|
| 1-16级 | 普通等级 |
| 15-16级法师 | **圣域**：领悟力、力量大幅提升 |
| 21级 | **传奇**：与封神独立的领域 |
| 封神后 | **神域** |

**职业体系**：
- **剑士**：七级后斗气可实体化，16级称号"圆桌骑士"，17级"圣骑士"
- **魔法师**：2:1上升，最高九阶魔法，16级进入圣域称"大魔导师"，18级死灵法师可转变为半神巫妖

## 主角人设

### 罗格

| 维度 | 内容 |
|------|------|
| 出身 | 破落贵族之子，死灵法师罗德格里斯的传人 |
| 性格 | **腹黑**——阴险狡诈，行事不择手段 |
| 野心 | "睡觉睡到自然醒，数钱数到手抽筋" |
| 战术风格 | 群殴是战略，暗算是战术 |
| 嗅觉 | 天生对金钱与权势极其敏锐 |

**身份最终形态**：精灵神使、阿雷公国公爵、阿斯罗菲克帝国亲王、神圣教会教皇、光明教会圣堂堂主

**成长轨迹**：
```
破落贵族
    ↓ 继承死灵法师灵魂与神之本源
创造会装死的骷髅风月
    ↓ 政治博弈、势力扩张
创立"战神之锤"
    ↓ 攀附奥菲罗克
诸神之战（风月死于诸神手下）
    ↓ 罗格爆发，冲向天界之门
位面战争：在虚无空间中开创属于自己的世界
```

## 爽点模型（暗黑风格）

| 爽点类型 | 触发场景 | 特点 |
|----------|----------|------|
| 逆袭爽点 | 小人物依靠智慧击败英雄 | 非力量碾压 |
| 权谋爽点 | 政治博弈层层算计 | 以弱胜强 |
| 后宫爽点 | 多位绝色女性 | 每段感情风格差异极大，富悲剧意味 |
| 力量爽点 | 从弱者到挑战诸神 | 成长感极强 |
| 独特宠物 | 风月（装死骷髅） | 辨识度极高的萌点 |

**核心冲突**：
> 当风月死于诸神手下时，罗格爆发——这是全书情绪最高点
> 不是热血沸腾，而是**愤怒与绝望中的疯狂**

## 文风特征

**语言风格**：
- 华丽而富有哲思
- 经典语录众多
- 示例：*"当光离开了暗，当海上有了天空，我惟愿以身躯化为大地，承载你疲累的双足，万载不变"*

**优点**：
- 设定出彩：脱胎于DND但不拘泥
- 人物刻画极为成功（罗格、风月、芙萝娅）
- 感情戏多元，每段感情风格差异极大
- 长线谋略：层层算计，环环相扣

**缺点**：
- 前期剧情较平
- 短线谋略不够精细
- 战争描写不足
- 部分支线挖坑不填

## 写作技巧总结

### 暗黑流主角塑造

**非脸谱化腹黑**：
```
表面：废物、贪财、好色
实际：心机深沉、算无遗策、关键时刻极度狠辣
```

### 权谋爽点写法

```
【普通爽文】VS【亵渎式爽点】
实力碾压        →  智谋以弱胜强
一拳打脸        →  布局多步后的全面崩溃
装逼打脸立刻兑现 → 长线伏笔回收
```

### 感情线设计

每段感情风格都不同：
- 风月：生死与共、灵魂羁绊
- 芙萝娅：妖精式的狡黠与危险
- 安德罗妮：命运的无奈与救赎
- 埃丽西斯：禁忌之恋的悲剧色彩
