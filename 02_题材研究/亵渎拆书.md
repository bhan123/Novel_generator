| 项目 | 内容 |

---

## 核心创新点

### 题材类型
**暗黑权谋西幻**——脱胎于DND但不拘泥，自成体系

**与小白文的本质区别**：
- 不走升级打怪换地图的套路
- 核心冲突是**权谋博弈**而非战斗力碾压
- 主角是小人物，靠智慧和手段而非无敌实力

---

## 世界观设定

### 位面体系

```
天界：十二主神主导的最强位面
    ↓
人界（格罗里亚大陆）：力量上限21级，故事主位面
    ↓
异界（死亡世界）：亡灵世界，八位最高君王主导
    ↓
魔界：魔族位面
    ↓
深渊世界：强大深渊生物生存
```

**核心规则**：
> "真正的力量是制订规则"
> 世界分为三级：神（掌控位面、攫取信仰之源）→ 强者（受限于位面规则）→ 凡人（最底层，作为"食粮"）

### 天使位阶体系

**上三级**：炽天使（四君主：米迦勒、加百列、拉斐尔、乌利尔）、智天使、座天使
**中三级**：主天使、力天使、能天使
**下三级**：权天使、大天使、天使

**特别说明**：光天使（如威娜）是不列入天使位阶的特殊存在，专为战斗设计，战斗力介于炽天使和智天使之间。堕落天使撒旦堕落前是六翼炽天使长拉斐尔。

## 修炼体系

| 等级 | 境界 |
|------|------|
| 1-16级 | 普通等级 |
| 15-16级法师 | **圣域**：领悟力、力量大幅提升 |
| 21级 | **传奇**：与封神独立的领域 |
| 封神后 | **神域** |

**职业体系**：
- **剑士**：七级后斗气可实体化，16级称号"圆桌骑士"，17级"圣骑士"
- **魔法师**：2:1上升，最高九阶魔法，16级进入圣域称"大魔导师"，18级死灵法师可转变为半神巫妖

## 详细剧情进展

### 【第一阶段：崛起篇】

#### 罗格如何获得死灵法师传承

| 项目 | 内容 |
|------|------|
| **身份** | 破落贵族之子，母亲偷情的产物 |
| **师承** | 十大魔导士之首——史上最伟大的死灵法师**罗德里格斯** |
| **继承经过** | 误打误撞之下，继承了罗德里格斯的灵魂与最纯净的灵魂能量：**神之本源** |
| **获得能力** | 强大无比的精神力、罗德里格斯指引他掌握力量的本质 |
| **最终结局** | 审判日后成功在自己的**绝对领域**内复活 |

**关键点**：罗德里格斯在**亵渎俄狄大神殿**后，利用光天使**威娜**的**神之本源**为罗格创造出来的宠物。

---

### 【第二阶段：风月创造与发展】

#### 风月创造过程

**初始形态**：一个小骷髅（罗德里格斯用威娜神之本源创造）

**成长历程**：
| 阶段 | 发展 |
|------|------|
| 初始 | 小骷髅形态 |
| 成长 | 在**死亡世界**逐步成长 |
| 成就 | 成为**第八位君王** |
| 收服 | 骨龙**格利高里**（原魔龙）作为手下 |
| 脱离 | 与威娜一同脱离死亡世界 |
| 成神 | 以女神**奥黛雷赫**之名创立**智慧之眼**教派，击败冰雪女神，吸收其神格，成为新的**冰雪女神** |

**与威娜的关系**：
- 两者**共用同一个神之本源**
- 在意识之海中**长期争斗**，争夺能量与控制权
- 天界及威本人都认为风月是威娜的分身

---

### 【第三阶段：女主详细剧情线】

#### 1. 风月（罗格最重要的女人）

| 项目 | 内容 |
|------|------|
| **初始身份** | 会装死的骷髅（罗德里格斯用威娜神之本源创造） |
| **最终身份** | 神圣教会女神、罗格的守护神 |
| **特殊之处** | 与威娜共用神之本源；罗格为她不惜牺牲一切 |
| **经典语录** | "你即已成神，那么我，从此也不再为人！" |
| **结局** | 与罗格互相拯救，先后将对方送入空间乱流，最终在罗格的精神领域内复活 |

#### 2. 芙萝娅（小妖精）

| 项目 | 内容 |
|------|------|
| **身份** | 莱茵同盟公主，绰号"小妖精" |
| **师承** | 十大魔导士之一的耶罗 |
| **能力** | 强大魔法师，精于医药炼毒，众多魔法道具 |
| **性取向** | 同性恋者，与安德罗妮是恋人关系 |
| **经历** | 国灭后投奔罗格 |
| **结局** | 为罗格坚持留在位面，**拒绝天使的拉拢从容赴死**（BE） |

#### 3. 安德罗妮

| 项目 | 内容 |
|------|------|
| **身份** | 罗歇里奥元帅之女，剑圣普罗西斯最有天赋的徒弟 |
| **持有神器** | 碧落星空 |
| **实力** | 圣域剑士 |
| **特殊情况** | 患有严重的**性别紊乱症**，初期以男性"安德烈"现身 |
| **性取向** | 同性恋者，后深爱上风月 |
| **结局** | 被**凯瑟琳暗杀**；遗愿是为罗格生一个儿子 |

#### 4. 埃丽西斯（魔族公主）

| 项目 | 内容 |
|------|------|
| **身份** | 冷艳高傲的魔族公主 |
| **地位** | 罗格内心深处的**挚爱** |
| **恋人** | 奥菲罗克（"黄金狮子"） |
| **结局** | 奥菲罗克为救她化为**堕落天使**，最终两人**同眠于圣焰中，化为第十座炼狱**（炼狱天使像） |

---

### 【第四阶段：诸神之战】

#### 主要神祇

| 阵营 | 神祇 |
|------|------|
| **光明诸神** | 至高神（老板）、迪斯马森（毁灭之主，十二主神首领）、以撒（救赎）、塞坦尼斯托利亚（战争）、席尔洛（欺骗谎言之主）、弗利安米尔（秩序）、提拉特弥斯（创造光天使者） |
| **黑暗神** | 黑暗主神巴萨罗狄摩高 |

#### 关键战役

**1. 奥菲罗克 vs 奥古斯都**：
- 黄金狮子为救爱人化为堕落天使
- 单挑血天使
- 最终折断奥古斯都翅膀

**2. 罗格 vs 魔皇**：
- 罗格为取得神格、解救风月与魔皇大战
- **最终杀死魔皇**

**3. 末日审判**：
- 天界占领罗格所在位面
- 信仰之源被破坏
- 整个位面**毁灭**

---

### 【第五阶段：结局】

**结局详情**：

1. **位面毁灭**：最后一战，天界占领罗格所在位面，信仰之源被破坏，整个位面毁灭

2. **逃脱**：罗格凭借自己领悟的**法则/绝对领域**逃出毁灭的世界

3. **开创世界**：罗格在虚无空间中**开创属于自己的世界**——其本质是虚无的存在，在罗格的意识和精神力下具象化

4. **复活**：罗格与风月在此位面内**"复活"**

5. **核心主题**：反抗者最终成为新秩序的创造者——这是对罗格此前所有恶行与反抗的**极大反讽**

## 主角人设

### 罗格

| 维度 | 内容 |
|------|------|
| 出身 | 破落贵族之子，死灵法师罗德格里斯的传人 |
| 性格 | **腹黑**——阴险狡诈，行事不择手段 |
| 野心 | "睡觉睡到自然醒，数钱数到手抽筋" |
| 战术风格 | 群殴是战略，暗算是战术 |
| 嗅觉 | 天生对金钱与权势极其敏锐 |

**身份最终形态**：精灵神使、阿雷公国公爵、阿斯罗菲克帝国亲王、神圣教会教皇、光明教会圣堂堂主

**成长轨迹**：
```
破落贵族
    ↓ 继承死灵法师灵魂与神之本源
创造会装死的骷髅风月
    ↓ 政治博弈、势力扩张
创立"战神之锤"
    ↓ 攀附奥菲罗克
诸神之战（风月死于诸神手下）
    ↓ 罗格爆发，冲向天界之门
位面战争：在虚无空间中开创属于自己的世界
```

## 爽点模型（暗黑风格）

| 爽点类型 | 触发场景 | 特点 |
|----------|----------|------|
| 逆袭爽点 | 小人物依靠智慧击败英雄 | 非力量碾压 |
| 权谋爽点 | 政治博弈层层算计 | 以弱胜强 |
| 后宫爽点 | 多位绝色女性 | 每段感情风格差异极大，富悲剧意味 |
| 力量爽点 | 从弱者到挑战诸神 | 成长感极强 |
| 独特宠物 | 风月（装死骷髅） | 辨识度极高的萌点 |

**核心冲突**：
> 当风月死于诸神手下时，罗格爆发——这是全书情绪最高点
> 不是热血沸腾，而是**愤怒与绝望中的疯狂**

## 文风特征

**语言风格**：
- 华丽而富有哲思
- 经典语录众多
- 示例：*"当光离开了暗，当海上有了天空，我惟愿以身躯化为大地，承载你疲累的双足，万载不变"*

**优点**：
- 设定出彩：脱胎于DND但不拘泥
- 人物刻画极为成功（罗格、风月、芙萝娅）
- 感情戏多元，每段感情风格差异极大
- 长线谋略：层层算计，环环相扣

**缺点**：
- 前期剧情较平
- 短线谋略不够精细
- 战争描写不足
- 部分支线挖坑不填

---

## 写作技巧总结

### 暗黑流主角塑造

**非脸谱化腹黑**：
```
表面：废物、贪财、好色
实际：心机深沉、算无遗策、关键时刻极度狠辣
```

### 权谋爽点写法

```
【普通爽文】VS【亵渎式爽点】
实力碾压        →  智谋以弱胜强
一拳打脸        →  布局多步后的全面崩溃
装逼打脸立刻兑现 → 长线伏笔回收
```

### 感情线设计

每段感情风格都不同：
- 风月：生死与共、灵魂羁绊
- 芙萝娅：妖精式的狡黠与危险
- 安德罗妮：命运的无奈与救赎
- 埃丽西斯：禁忌之恋的悲剧色彩
