## 三、平台对比分析

### 3.1 核心差异

| 维度 | 起点中文网 | 番茄小说 |
|------|----------|----------|
| **主流基调** | 修仙/仙侠为王，灵异辅助 | 都市/异能为主，规则怪谈崛起 |
| **读者画像** | 老白读者多，偏好深度、凡人流 | 年轻读者多，偏爱脑洞、快节奏爽文 |
| **题材偏好** | 凡人流、家族修仙、正统修真 | 规则怪谈、都市异能、上交国家 |
| **内容风格** | 世界观宏大、文笔考究、有深度 | 节奏快、脑洞大、爽点密集 |
| **IP方向** | 影视为主（《捞尸人》亿元影视改编） | 动画/短剧为主（《斩神》动画破7亿） |
| **互动模式** | 传统订阅制，月票榜 | 读者共创+短视频传播 |
| **作者生态** | 大神效应强，老牌白金多 | 新人出头快，60万签约作者 |

### 3.2 共同趋势

| 趋势 | 说明 |
|------|------|
| **穿越是标配** | 两平台爆款绝大多数带穿越元素 |
| **系统面板流行** | "百世书"系统、怪物收容监狱、双鱼玉佩等设定受捧 |
| **都市+异能** | 都市背景结合异能/超自然是大方向 |
| **灵异/悬疑走强** | 起点捞尸人、番茄诡舍，民俗+现代悬疑融合 |
| **读者参与深化** | 从被动阅读到主动影响剧情走向 |
| **短视频联动** | 抖音二创成为重要流量来源 |
| **IP多元化** | 影视、动画、短剧多形态改编 |

## 四、结论与建议

### 4.1 关键发现

1. **起点**：修仙是基本盘，凡人流复兴，灵异破圈，科幻增长；老白读者主导，大神效应显著
2. **番茄**：都市异能+规则怪谈双引擎，上交国家题材独有，短视频传播强，IP动画化快
3. **共性**：穿越+系统是标配，灵异/悬疑走强，读者参与创作趋势明显
4. **差异**：起点偏"深度/修仙"，番茄偏"脑洞/都市"；起点看榜单，番茄看话题

### 4.2 创作建议

| 方向 | 起点适配 | 番茄适配 |
|------|---------|---------|
| 凡人修仙+家族群像 | ✅ 最强赛道 | △ 需改编 |
| 民俗灵异+现代背景 | ✅ 破纪录赛道 | ✅ 诡舍验证 |
| 规则怪谈+读者投票 | △ 起点适合但不主打 | ✅ 现象级赛道 |
| 上交国家+修仙 | △ 适合但非主流 | ✅ 番茄特色 |
| 都市异能+爽文 | △ 辅助赛道 | ✅ 基本盘 |
| 科幻+高武 | ✅ 增长赛道 | △ 需结合其他 |
